# cnki-PDF-download

一款适配中国知网（CNKI）的浏览器插件，可提供便捷的论文 PDF 下载与阅读体验。

</p>

## 说明

### 功能特性

#### 📥 PDF 下载

- 为每篇文献添加「下载」按钮
- 支持期刊论文和学位论文 PDF/CAJ 下载
- 自动识别 DOI 链接、来源链接等多种下载渠道

#### 📖 摘要悬停预览

- 鼠标悬停文献标题即可查看摘要内容
- 自动加载并格式化显示
- 支持显示关键词信息（如有）

#### 📦 批量下载管理

- 添加「批量下载」按钮，支持批量选择文献
- 智能排队下载，可调节下载间隔
- 实时显示下载进度和状态统计
- 支持暂停/继续、取消操作
- 自动检测验证码并提示用户手动验证

### 安装方法

#### Chrome/Edge 浏览器

1. 在浏览器地址栏输入 `edge://extensions/`（Edge）或 `chrome://extensions/`（Chrome）
2. 开启右上角「开发人员模式」
3. 点击「加载已解压的扩展程序」
4. 选择插件所在文件夹
5. 启用插件

### 使用说明

1. **单篇下载**：在搜索结果页面，每篇文献标题旁会显示「下载」按钮，点击即可下载
2. **摘要预览**：将鼠标悬停在文献标题上，会弹出提示框显示摘要和关键词
3. **批量下载**：点击页面底部的「批量下载」按钮，在弹出面板中选择要下载的文献，点击「开始下载」即可

### 注意事项

- 使用前请确保已登录 CNKI 账号并拥有相应文献的下载权限
- 建议适当设置下载间隔（默认 8 秒），避免触发反爬限制
- 遇到验证码时，请手动完成验证后点击「继续」按钮
