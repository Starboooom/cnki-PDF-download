# cnki-PDF-download

A browser extension for CNKI (China National Knowledge Infrastructure) that provides convenient PDF download and reading experience.

## Introduction

### Features

#### 📥 PDF Download

- Add "Download" button for each article
- Support PDF/CAJ download for journal articles and dissertations
- Auto-detect DOI links, source links, and various download channels

#### 📖 Abstract Hover Preview

- View abstract content by hovering over article titles
- Auto-load and format display
- Support keyword display (if available)

#### 📦 Batch Download Management

- Add "Batch Download" button for selecting multiple articles
- Intelligent queued downloading with adjustable delay
- Real-time download progress and status statistics
- Support pause/resume and cancel operations
- Auto-detect captcha and prompt user for manual verification

### Installation

#### Chrome/Edge Browser

1. Open `edge://extensions/` (Edge) or `chrome://extensions/` (Chrome)
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked"
4. Select the extension folder
5. Enable the extension

### Usage

1. **Single Download**: Click the "Download" button next to each article title
2. **Abstract Preview**: Hover over article titles to view abstract and keywords
3. **Batch Download**: Click "Batch Download" button at the bottom of the page, select articles, and click "Start Download"

### Notes

- Ensure you are logged in to CNKI and have proper download permissions
- Recommended download interval is 8 seconds (default) to avoid anti-crawl restrictions
- When captcha appears, complete verification manually and click "Resume"
